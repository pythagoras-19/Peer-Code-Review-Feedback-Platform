# Peer Code Review & Feedback Platform

A Next.js + TypeScript frontend for peer code review with Supabase authentication.

## Quick Start

```bash
npm install
cp .env.local.example .env.local
# Add your Supabase URL and anon key to .env.local
npm run dev
```

Get Supabase credentials from: https://app.supabase.com → Project → Settings > API

## Features

- Email/password authentication (login & signup)
- Client-side session management
- Form validation and error handling
- Responsive UI with clean styling

## Routes

- `/` - Home page
- `/login` - Login page
- `/signup` - Sign up page
